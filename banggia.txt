Lưu ý:

================================================================
1) PHẦN MỀM DESKTOP
================================================================
Basic
- Đặc điểm:
  + 1-3 chức năng chính
  + Giao diện đơn giản
  + Lưu dữ liệu local
  + Không cần login
  + Không kết nối server
  + Code sạch, chạy ổn định
- Ngày: 1-2 ngày
- Giá: 700K - 1.500K

Standard
- Đặc điểm:
  + 3-5 chức năng (CRUD đầy đủ)
  + UI rõ ràng, dễ dùng
  + Có database (MySQL/SQLite)
  + Có login cơ bản
  + Có validate dữ liệu
  + Xuất file (Excel/PDF cơ bản)
- Ngày: 3-5 ngày
- Giá: 2.000K - 4.000K

Advanced
- Đặc điểm:
  + 6+ chức năng (full nghiệp vụ)
  + Phân quyền (admin/user)
  + Database chuẩn (MySQL/PostgreSQL)
  + Backup dữ liệu
  + UI custom đẹp hơn
  + Tối ưu hiệu năng, có thể tích hợp API
- Ngày: 1-2 tuần
- Giá: 5.000K+

2) LANDING PAGE
================================================================
Basic
- Đặc điểm:
  + 1 trang duy nhất
  + HTML/CSS cơ bản
  + Có form liên hệ
  + Responsive cơ bản
- Ngày: 1 ngày
- Giá: 500K - 800K

Standard
- Đặc điểm:

  + Responsive chuẩn mobile
  + Hiệu ứng scroll animation
  + Tối ưu tốc độ

- Ngày: 2-3 ngày
- Giá: 1.000K - 2.300K

Advanced
- Đặc điểm:
  + Chuẩn SEO (meta, schema)
  + Tối ưu conversion (CTA)
  + Animation nâng cao
  + Kết nối Facebook Pixel/Google Analytics
  + Load nhanh dưới 3s
- Ngày: 3-5 ngày
- Giá: 2.700K+

3) WEBSITE THƯƠNG MẠI
================================================================
Basic
- Đặc điểm:

  + Giỏ hàng đơn giản
  + Không login
  + Admin chỉnh sản phẩm

  + API backend riêng
  + Bảo mật + tối ưu
  + Dashboard thống kê
- Ngày: 2-4 tuần
- Giá: 15.000K+

5) TOOL SCRAPING DATA
================================================================
Basic
- Đặc điểm:
  + Crawl 1 website (tĩnh, anti-bot nhẹ)
  + Lấy dữ liệu cơ bản: tên, giá, link
  + Xuất Excel/CSV
  + Chạy 1 lần
- Ngày: 1-2 ngày
- Giá: 500K - 1.500K

Standard
- Đặc điểm:
  + Crawl nhiều trang (pagination)
  + Lọc dữ liệu theo yêu cầu
  + Xử lý data (loại trùng, format)
  + Xuất Excel đẹp (cột rõ ràng)
  + Có script chạy lại
- Ngày: 2-5 ngày
- Giá: 2.000K - 5.000K

Advanced
- Đặc điểm:
  + Crawl nhiều website
  + Bypass cơ bản (anti-bot nhẹ)
  + Tự động chạy theo lịch
  + Lưu database (MySQL)
  + API lấy dữ liệu
  + Dashboard xem data
- Ngày: 1-2 tuần
- Giá: 6.000K - 15.000K+

- Ngày: 1-2 ngày

- Đặc điểm:
  + Xử lý nhiều bước logic

  + Đọc dữ liệu từ Excel
  + Có log hoạt động
  + Xử lý lỗi cơ bản

- Giá: 3.000K - 7.000K

Advanced
- Đặc điểm:
  + Điều khiển nhiều browser
  + Proxy/đổi IP
  + Kịch bản phức tạp
  + GUI điều khiển
  + Chạy nền/scheduler
  + Tích hợp scraping + automation
- Ngày: 1-3 tuần
- Giá: 8.000K - 20.000K+
